# feature-requirements-mvp

## Локальная разработка

### Настройка таймаута для локальной разработки

Таймаут функции установлен на **180 секунд (3 минуты)** в `netlify.toml` для продакшена.

Для локальной разработки с `netlify dev` по умолчанию используется таймаут 30 секунд. Чтобы увеличить его до 3 минут:

**Автоматическая настройка (рекомендуется):**
```bash
./setup-timeout.sh
```

Скрипт автоматически найдет и модифицирует файл Netlify CLI, установив таймаут на 3 минуты (180000 мс).

**Ручная настройка:**
1. Найдите файл: `<path_to_netlify_cli>/src/utils/dev.ts`
2. Найдите константу `SYNCHRONOUS_FUNCTION_TIMEOUT`
3. Измените значение на `180000` (180 секунд = 3 минуты)

**Важно**: После обновления Netlify CLI изменения будут потеряны. Запустите `./setup-timeout.sh` снова после обновления.

### Запуск

```bash
netlify dev
```
